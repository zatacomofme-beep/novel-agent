## 一、核心写作流程重构

### 1.1 三阶段创作模型

```
第一阶段：想法 → 大纲生成
第二阶段：细纲 + 写作意图确认
第三阶段：Agent 集群生成 → 审阅循环
```

### 1.2 流程详情

#### 第一阶段：创意输入 → 大纲生成

| 步骤 | 说明 |
|-----|------|
| 用户输入 | 用户直接输入想法（一句话/一段话） |
| AI 生成 | 系统自动生成三级大纲（章节概要） |
| 用户审阅 | 用户确认/修改大纲 |
| 进入下一阶段 | 大纲确认后进入第二阶段 |

#### 第二阶段：细纲 + 写作意图确认

| 步骤 | 说明 |
|-----|------|
| 用户细化 | 用户在三级大纲基础上细化 |
| AI 阐述意图 | LLM 阐述本章写作意图（100-300 字），说明计划怎么写 |
| 用户确认 | 用户确认写作意图后开始写，或直接修改意图后再生成 |

**写作意图示例**：
> "本章计划从主角被迫离开家乡切入，通过回忆展现师徒羁绊，为后续背叛埋线。采用倒叙与现实交织的手法，前半段压抑蓄力，后半段爆发冲突。"

#### 第三阶段：Agent 集群生成 → 审阅循环

| 步骤 | 说明 |
|-----|------|
| Agent 执行 | Agent 集群并行工作，生成章节初稿 |
| 用户审阅 | 用户审阅生成的章节 |
| 两种分支 | OK → 通过并保存 / 不OK → 进入修订流程 |

**修订流程**：
```
不通过
   ├── 手动直接改（用户自行编辑正文）
   └── 告诉 Agent 反馈 → 重新生成 → 再审阅（可循环）
```

## 二、设定圣经模块重构

### 2.1 核心变化

| 维度 | 当前（被动模式） | 新（主动模式） |
|-----|-----------------|---------------|
| 设定来源 | 从章节生成时自动提取 | 用户主动设计/生成 |
| 约束力 | 无 | 生成章节时必须参考 |
| 生成时机 | 章节副产品 | 可独立主动生成 |

### 2.2 设定分类

| 类别 | 内容 |
|-----|------|
| 世界基底 | 世界类型、核心规则、基础设定 |
| 物理世界 | 地理、时间规则、空间规则 |
| 力量体系 | 力量来源、等级划分、代价限制、传承方式 |
| 社会结构 | 势力格局、阶层流动、文化价值观 |
| 人物设定 | 主角/配角/反派的外貌、性格、背景、动机 |
| 物品设定 | 神器、道具、信物、功法、丹药 |
| 势力设定 | 宗门、家族、组织、联盟 |
| 剧情线设定 | 主线、支线、伏笔、核心悬念 |

### 2.3 冲突检测

- 章节内容与圣经设定矛盾时，主动提示用户
- 生成章节前校验设定一致性
- 优先使用圣经设定，章节内容服从圣经

## 三、引导式设定生成流程

### 3.1 设计原则

1. **递进式提问** - 从宏观到微观，从抽象到具体
2. **模型主动扩展** - 用户给一个方向，模型自己联想填充
3. **可跳过/可补充** - 不是强制流程，用户可以跳到喜欢的部分
4. **最终汇聚** - 所有设定汇总为结构化世界观文档

### 3.2 模型选择

- **使用模型**：`gemini-3.1-pro-preview`（通过 model_gateway 调用）
- **Provider**：`openai-compatible`（云雾 API 网关）

### 3.3 八步引导流程

---

#### 第一问：世界基底（必答）

**初始提问**：
```
请问，这是一个什么样的世界？
```

**模型扩展（基于用户回答自动进行）**：
```
你描述了一个有[宗门、灵根、飞升体系]的修仙世界。
那我想进一步了解：

1. 灵根是天生的还是后天可以获取？
2. 飞升是唯一目标还是有其他追求？
3. 宗门之外还有其他势力吗？（散修、魔道、世俗皇朝...）
4. 这个世界的历史有多久？有没有断层或浩劫？
```

---

#### 第二问：物理世界

**提问内容**：
```
基于你描述的世界，我想了解物理层面：

1. 这个世界的地理是什么样的？
   - 有几块大陆？海洋/星空/异空间的比重？
   - 灵气/能量分布均匀还是极偏？

2. 时间规则是什么样的？
   - 有没有季节/昼夜特殊规则？
   - 修炼是否有时限？（瓶颈、寿元限制）

3. 空间规则有什么特殊之处？
   - 是否有秘境/小世界/折叠空间？
   - 距离用什么衡量？（里/灵气浓度/传送阵？）
```

---

#### 第三问：力量体系

**提问内容**：
```
这个世界有怎样的力量系统？

1. 力量的来源是什么？
   - 灵气/斗气/查克拉/神力/科技？

2. 力量的等级如何划分？
   - 有多少阶层？名称是什么？
   - 升级是靠自己、靠资源、靠顿悟？

3. 力量有什么代价或限制？
   - 使用过度会反噬？消耗寿命？有冷却？

4. 力量能否传承？
   - 功法、血脉、器物、记忆？

5. 特殊体质/天赋是否存在？
   - 灵根/血脉/天生异能？
```

---

#### 第四问：社会结构

**提问内容**：
```
这个世界的社会是如何组织的？

1. 势力格局
   - 主要有哪些势力？（宗门、家族、帝国、散修联盟...）
   - 势力之间是什么关系？（联盟、对立、中立？）

2. 阶层流动
   - 普通人能否逆天改命？
   - 底层逻辑是什么？（血脉、天赋、资源、机缘？）

3. 核心规则
   - 法律/弱肉强食/潜规则？
   - 有什么是绝对禁忌？（同门相残、魔道手段...）

4. 文化与价值观
   - 这个世界的修士追求什么？（长生、力量、守护、逍遥...）
   - 有什么独特的风俗或信仰？
```

---

#### 第五问：主角定位

**提问内容**：
```
你打算写谁的故事？

1. 主角是什么存在？
   - 人类/妖兽/器灵/重生者/穿越者/本土生灵？

2. 主角的起点是什么？
   - 废物流？天才流？大家族？流浪儿？
   - 故事开始时处于什么境界/处境？

3. 主角的核心动机是什么？
   - 复仇？崛起？守护？探索？自我实现？

4. 主角有什么独特之处？
   - 特殊体质/背景/机缘/缺陷？

5. 主角的成长轨迹倾向？
   - 稳扎稳打/越级战斗/机缘巧合型？
```

---

#### 第六问：核心冲突

**提问内容**：
```
故事的核心矛盾是什么？

1. 主要冲突
   - 主角要面对的核心矛盾？（生存威胁/宿敌/天道不公/自我挣扎...）
   - 对立面是谁？（人、势力、命运、本心...）

2. 核心悬念
   - 贯穿全文的悬念？（最终boss身份/隐藏真相/主角身世...）
   - 中短期的目标？（近期要解决的事件）

3. 情感基调
   - 整体是热血/压抑/轻松/黑暗？
   - 有没有虐点？（牺牲、背叛、遗憾...）

4. 爽点设计
   - 你想要的爽感类型？（逆袭、打脸、智斗、热血团战...）
```

---

#### 第七问：世界细节（可选）

**提问内容**：
```
还有一些细节可以补充：

1. 地理地标
   - 有哪些重要地点需要提前设定？（宗门山门、禁地、古战场...）

2. 重要势力详情
   - 主角所属势力的定位？（首徒/宗主之女/叛徒...）
   - 敌对势力的风格？（正道伪君子/魔道真小人/中立势力...）

3. 核心物品
   - 有没有重要器物/功法/丹药需要设定？（本命法宝/传承之物...）

4. 特殊设定
   - 有没有这个世界的独特规则？（丹药有毒/剑灵意识/器灵修炼...）
```

---

#### 第八问：风格偏好（可选）

**提问内容**：
```
最后，想确认一些风格偏好：

1. 文风倾向
   - 简洁干脆/细腻华丽/大白话？
   - 战斗描写偏多还是心理描写偏多？

2. 节奏偏好
   - 快节奏（升级快/冲突密集）
   - 慢热型（铺垫长/伏笔多）

3. 参考作品
   - 有没有对标的作品？（《斗破苍穹》/《凡人修仙传》/《诡秘之主》...）

4. 规避事项
   - 有没有什么不想写的桥段？（送女/舔狗/无脑打脸...）
```

## 四、关键设计要点

| 设计点 | 说明 |
|-------|------|
| **模型主动扩展** | 不只是记录用户回答，还要主动联想和追问 |
| **递进但不强制的顺序** | 问题是递进的，但用户可以跳到自己想设计的部分 |
| **记忆上下文** | 模型记住用户之前的回答，保证设定自洽 |
| **中途可退出** | 用户可以随时退出，已设定的内容自动保存 |
| **可视化汇总** | 最后有一个人类可读的世界观文档 |
