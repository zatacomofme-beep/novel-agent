# 文案风格指南

> 这是整个产品的文字规范，定义了术语、语气、按钮、提示语等所有面向用户的文字。
> 所有新增或修改的文案必须符合本指南。

---

## 一、核心定位：写作陪伴者

- 不是"AI内容生成工具"，而是"懂你写作节奏的创作伙伴"
- 语气温暖、鼓励、专业，像一个经验丰富的编辑在旁边
- 引导而非命令，给选择而非给限制

---

## 二、语气原则

| 原则 | 说明 | 示例 |
|------|------|------|
| 用"你"不用"用户" | 平等交流 | ✅ "你还没起稿呢" / ❌ "用户尚未创建章节" |
| 用"我们"不用"系统" | 陪伴感 | ✅ "我们来写这一章" / ❌ "系统正在生成" |
| 动词优先 | 有行动感 | ✅ "开始下一章" / ❌ "进入下一章节" |
| 不命令 | 温和引导 | ✅ "在这里写下想法" / ❌ "请输入内容" |
| 不暴露技术 | 用户看不见技术细节 | ✅ "智能续写中" / ❌ "调用 Gemini API" |
| 具体而非模糊 | 告诉用户能做什么 | ✅ "最多写2000字" / ❌ "字数超限" |

## 三、术语表（概念统一）

| 概念 | 我们用的词 | 不用的词 |
|------|-----------|---------|
| 用户写的章节正文 | **正文** / **章节** | chapter_content, content |
| AI续写的内容 | **续写稿** | draft, generated_text |
| 章节结构规划 | **章纲** | outline（三级都叫章纲） |
| 确认后的最终版本 | **终稿** | final_draft |
| 创作主页面 | **创作空间** / **故事屋** | story-room |
| 设定存储库 | **设定本** | story-bible, bible |
| 章节写作思路 | **写作意图** | writing_intent |
| 可以跳过的步骤 | **可不填** | optional（中文语境） |
| 章节版本 | **历史版本** | version history |
| 大纲结构 | **章纲** / **章节骨架** | outline |
| 生成失败重试 | **再试一次** | retry |
| 内容不符合预期 | **方向不对** | low quality |

## 四、页面文案规范

### 4.1 按钮文字

| 场景 | 建议文字 | 备注 |
|------|---------|------|
| 开始一个新章节 | **"开始这一章"** 或 **"起稿"** | "起稿"更有写作者气质 |
| AI续写一段 | **"帮我续一段"** | 主动、亲切 |
| 保存当前内容 | **"收下这段"** | 确认感，像收藏 |
| 回退到上一版 | **"用前一个版本"** | 具体，知道自己在做什么 |
| 清空续写内容 | **"不用这个版本"** | 温和否定，不浪费 |
| 查看历史版本 | **"看看之前写过什么"** | 好奇感 |
| 基于指令重写 | **"按这个方向重写"** | 有掌控感 |
| 新建项目 | **"开一本新书"** | 写手的语言 |
| 应用风格模板 | **"套用这个手感"** | 有温度 |
| 提交审核 | **"送审"** | 简洁行业语 |
| 采纳优化建议 | **"采纳"** | 单字有力 |

### 4.2 空状态文案

| 场景 | 建议文字 |
|------|---------|
| 还没有章节 | "还没有正文，先来起个头吧" |
| 还没有章纲 | "还没有章纲，先来规划一下这一章写什么？" |
| 还没有设定本 | "设定本还是空的，先来搭好世界观" |
| 还没有风格模板 | "还没有锁定手感，去风格中心挑一个？" |
| 还没有历史版本 | "这是第一个版本，写下去就有历史了" |
| 还没有协作者 | "还没有其他伙伴，可以邀请合作者" |

### 4.3 加载中文案

| 场景 | 建议文字 |
|------|---------|
| 正在构思章纲 | "正在构思中…" |
| 正在续写正文 | "AI 正在写下一段…" |
| 正在生成章纲 | "AI 正在规划章节结构…" |
| 正在保存 | "正在收下…" |
| 正在加载设定本 | "正在翻设定本…" |

### 4.4 成功文案

| 场景 | 建议文字 |
|------|---------|
| 保存成功 | "已经收下了" / "这一段已经在你手上了" |
| 版本已保存 | "版本已锁定，这一版不会再变了" |
| 终稿确认成功 | "终稿已锁定，这一章完成了" |
| 风格模板已应用 | "这套手感已经套进整本书了" |
| 设定已保存 | "设定本已更新" |

### 4.5 错误文案

| 场景 | 建议文字 | 补充说明 |
|------|---------|---------|
| 网络错误 | "网络好像不太顺，试试再来一次？" | 给出重试指引 |
| 生成失败 | "这次没写顺，试试调整下章纲再来？" | 给出调整方向 |
| 保存失败 | "收下的时候出了点小状况，没有丢内容，再试一次？" | 确认内容没丢 |
| 章节不存在 | "这一章不见了，刷新看看？" | 引导刷新 |
| 未检测到写作意图 | "先说说这一章想写什么，AI 才知道怎么帮你" | 引导补全 |

### 4.6 引导提示文案

| 场景 | 建议文字 |
|------|---------|
| 输入章纲 | "写一段话，描述这一章要发生什么" |
| 输入写作意图 | "用一两句话说说这一章的核心目标，比如'主角第一次战斗'" |
| 输入续写提示 | "告诉 AI 你想要什么，比如'再来一段打斗'" |
| 章节数设置 | "告诉 AI 你计划写多少章，帮助它规划节奏" |
| 选择风格模板 | "挑一个和你想要的文风最像的，AI 会按这个感觉来写" |

## 五、世界观构建8步引导文案

| 步骤 | 标题 | 用户看到的引导语 |
|------|------|----------------|
| 1 | 世界基底 | "请问，这是一个什么样的世界？" |
| 2 | 物理世界 | "这个世界的物理法则和自然环境有什么特别之处？" |
| 3 | 力量体系 | "这个世界存在什么样的力量体系或超凡能力？" |
| 4 | 社会结构 | "这个世界的社会结构、势力划分和主要矛盾是什么？" |
| 5 | 主角定位 | "你的主角是什么人？有什么独特之处？" |
| 6 | 核心冲突 | "故事的核心冲突是什么？主角面临的主要挑战是什么？" |
| 7 | 世界细节 | "还有什么想要补充的世界设定吗？如历史、文化、地理等。（可不填）" |
| 8 | 风格偏好 | "你希望故事的整体风格是什么样的？轻松、压抑、热血？（可不填）" |

```
404 / 500 / 503
agent / coordinator / gateway
chunk / paragraph
schema / model / entity
SQL / Redis / Celery
```
