## 模板数据结构

每套模板包含：
- **基本信息**: 名称、简介、描述、分类、难度
- **内容**: 详细的写作框架和指导（支持 Markdown）
- **变量**: 可动态替换的占位符
- **标签**: 便于搜索和推荐
- **推荐场景**: 适用场景列表

## 如何导入完整模板

### 方法 1: 自动导入（推荐）

系统会在首次访问时自动导入所有模板。无需手动操作。

访问：http://localhost:3000/dashboard/prompt-templates

### 方法 2: 运行种子脚本

```bash
cd /Users/libenshi/Desktop/novels/backend
python scripts/seed_templates.py
```

### 方法 3: 手动创建

通过前端界面逐个创建 24 套模板（不推荐，太耗时）

## 使用流程

1. **访问模板库**: http://localhost:3000/dashboard/prompt-templates
2. **浏览/搜索模板**: 按分类、难度、标签筛选
3. **查看详情**: 点击模板查看内容、变量、示例
4. **应用模板**: 点击"应用到当前章节"
5. **自动跳转**: 进入 story-room 开始写作
6. **自定义**: 可创建自己的模板
